# Projects Index

This folder contains six sequential IAM projects, each documenting a phase of building out the TechMigos Enterprise IAM Lab.

1. **[01-okta-organization-setup](./01-okta-organization-setup)** — Core Okta tenant configuration, the foundation everything else builds on.
2. **[02-profile-editor-and-attribute-mapping](./02-profile-editor-and-attribute-mapping)** — Custom attributes and mapping validation.
3. **[03-user-lifecycle](./03-user-lifecycle)** — Automated joiner-mover-leaver rules.
4. **[04-saml-salesforce](./04-saml-salesforce)** — SAML SSO federation to Salesforce.
5. **[05-saml-zendesk](./05-saml-zendesk)** — SAML SSO federation to Zendesk.
6. **[06-security-policies](./06-security-policies)** — Authentication and access policy configuration.

Each project folder contains its own README with objectives, steps taken, and screenshots.
